<h1>Managing Groups in Okta</h1>

This tutorial outlines the process of creating groups and automating user assignments using Group Rules within an Okta tenant.<br />

<h2>Environments and Technologies Used</h2>

- Okta Integrator Free Plan

<h2>Operating Systems Used</h2>

- Windows 11

<h2>Group Management Tasks</h2>

- Created HR Group
- Created IT Group
- Created Sales Group
- Assigned users to groups
- Configured Group Rules
- Automated user-to-group assignments
